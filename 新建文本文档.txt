NetWorkGuiManager.addGifURL(String url) 添加一个gif图片地址到地址集合中(客户端进入时将会下载集合内所有图片)
NetWorkGuiManager.addImageURL(String url) 添加一个图片地址到地址集合中(客户端进入时将会下载集合内所有图片)
若要使用网络图片以上两步必须执行
NetWorkGui nwg = new NetWorkGui(int id) 以id生成一个GUI
NetWorkGuiManager.distributeID() 获取一个不重复的id
NetWorkGui.objList.add(NetWorkGuiObject obj) 向GUI中添加组件
API.openGui(Player p, NetWorkGui nwg) 为玩家打开GUI
API.closeNowGui(Player p) 为玩家关闭现在的GUI
NetWorkGuiButton(String str, int id, int x, int y, int width, int height) 构造一个按钮
NetWorkGuiCustomButton(String str, String firstUrl, String lastUrl, int id, int x, int y, int width, int height, int firstColor, int lastColor) 构造一个自定义按钮
firstUrl为鼠标未悬停时的图片
lastUrl为鼠标悬停时的图片
NetWorkGuiCustomField(String url, int id, int x, int y, int width, int height, int textureX, int textureY, int maxStringLength) 构造一个自定义输入框
NetWorkGuiField(int id, int x, int y, int width, int height, int maxStringLength) 构造一个输入框
NetWorkGuiImage(String path, int id, int x, int y, int textureX, int textureY, int width, int height) 构造一个图片
NetWorkGuiString(String str, int id, int x, int y, int color) 构造一个文本
NetWorkGuiUrlGif(String path, int id, int speed, int x, int y, int textureX, int textureY, int width, int height) 以单个url构造gif图片
speed越小速度越快
NetWorkGuiUrlsGif(String[] paths, int id, int speed, int x, int y, int textureX, int textureY, int width, int height) 以多个url构造gif图片
speed越小速度越快
